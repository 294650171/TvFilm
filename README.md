# TvFilm
创建
##安卓电视app
 * 针对安卓电视项目的焦点处理，抽取的baseactivity，及MainActivity中onKeyDown方法的处理。
 * 选择器、动画放大的特效。
 * 项目架构采取MVP,感谢MVPHelper插件的作者降低新建类的工作量。
 * 后端数据采用的是WebSocket，封装的androidasync库，并使用接口对返回值进行处理。
